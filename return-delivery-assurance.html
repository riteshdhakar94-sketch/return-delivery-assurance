Return Delivery Assurance

Tracks return shipments across two reconciliation stages:

1. Expected vs Delivered — Compares returns scheduled for Out for Delivery (OFD) on a given date against what the courier actually delivered. A gap is a courier shortfall.

2. Delivered vs Received — Compares courier-delivered returns against what the warehouse physically acknowledged. A gap is a warehouse receipt shortfall.

OFD data is a pre-shipment manifest uploaded per courier partner, indicating shipments scheduled for delivery on a given date. If not uploaded, expected counts will be understated and delivered quantities may appear to exceed 100%.
